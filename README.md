## 기술 스택

* Language : Dart
* framework : Flutter
* IDE : Visual Studio Code

### 기능
* CRUD 로직은 다 구현했지만 시간 부족으로 R(읽기) 기능만 적용
* 메인 페이지에서 앱바 내 액션 버튼을 구현해서 Creat기능 적용 하지만 시간과 역량 부족으로 기능 구현 실패
* 메인 페이지 텍스트 버튼을 누르게 되면 main_service.dart에 있는 데이터들이 서브페이지 텍스트필드 안으로 넘김
